## Features

- **Real-time Weather Data**: Get current conditions for any city worldwide
- **Multi-day Forecast**: Interactive temperature trend chart showing 5-day forecast
- **Clean UI**: Responsive design that works on desktop and mobile
- **Error Handling**: Robust error handling for API failures and invalid cities
- **API Integration**: Uses OpenWeatherMap API for accurate weather data

## Setup

1. Get an API key from [OpenWeatherMap](https://openweathermap.org/api)
2. Set the `OWM_API_KEY` environment variable in Project Settings
3. Open `index.html` in your browser or deploy to a web server

## Usage

1. Enter a city name in the search field
2. Optionally provide an API key if not set as environment variable
3. Click "Search Weather" to get current conditions and forecast
4. View the interactive temperature chart showing trends over time

## Technology Stack

- **Frontend**: Vanilla HTML5, CSS3, JavaScript (ES6+)
- **Charts**: Chart.js for data visualization
- **API**: OpenWeatherMap REST API
- **Backend**: Next.js API routes (for server-side API calls)
